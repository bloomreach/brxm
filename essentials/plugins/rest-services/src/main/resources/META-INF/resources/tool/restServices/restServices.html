REST resources

The Content REST API enables you and your users to access the content in the repository through generic REST resources. If you are a new Hippo user and you want to experiment with Hippo's capabilities, this is the recommended mechanism to use. It can be combined with manual resources, which provide greater control over the look-and-feel and capabilities of your resources, but requires more Java knowledge and some manual steps.

Choose the base URL for the generic REST resources

http://localhost:8080/site/

The following REST resource will be available: http://localhost:8080/site/{{genericRestName}}/documents

Choose the base URL for the manual REST resources

http://localhost:8080/site/

Select document type(s) to expose through REST
